# Akademi_GIB - Sınav Hazırlık Uygulaması

Python Kivy ile yapılmış mobil sınav hazırlık uygulaması.

## Özellikler
- Kullanıcı girişi ve kaydı
- Soru bankası (çoktan seçmeli)
- 1 dakikalık soru süresi
- İlerleme takibi
- Puan hesaplama

## Kurulum

```bash
pip install -r requirements.txt
python main.py
```

## Proje Yapısı
- `main.py` - Ana uygulama dosyası
- `database.py` - Veritabanı işlemleri
- `screens/` - Ekranlar
- `models/` - Veri modelleri
- `utils/` - Yardımcı fonksiyonlar
